# Dataset Description

## Dataset
Breast Cancer Wisconsin Diagnostic Dataset

Source:
UCI Machine Learning Repository  
https://archive.ics.uci.edu/ml/datasets/Breast+Cancer+Wisconsin+(Diagnostic)

## Overview

The dataset contains diagnostic measurements of breast cancer cell nuclei extracted from digitized images of fine needle aspirates of breast masses.

Each instance in the dataset corresponds to a patient record with several computed features describing characteristics of the cell nuclei.

## Dataset Size

Number of instances: 569  
Number of features: 30  
Target variable: Diagnosis (Malignant or Benign)

## Feature Examples

Some of the diagnostic features include:

- radius
- texture
- perimeter
- area
- smoothness
- compactness
- concavity
- symmetry

These features are used as input variables for machine learning classification models.

## Target Variable

The dataset contains a binary classification target:

- M = Malignant
- B = Benign

## Usage in This Project

The dataset was used to train and evaluate machine learning classification models for breast cancer diagnosis using the Scikit-learn library.
