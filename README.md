# Breast Cancer Diagnostic Machine Learning Model

This repository contains a machine learning project for breast cancer diagnosis using the Breast Cancer Wisconsin Diagnostic dataset from the UCI Machine Learning Repository.

## Project Overview

The objective of this project is to explore the use of machine learning techniques for the classification of breast cancer tumors as benign or malignant based on diagnostic measurements derived from digitized images of fine needle aspirates of breast masses.

The project includes data preprocessing, feature analysis, model training, and evaluation using standard classification metrics.

## Dataset

Breast Cancer Wisconsin Diagnostic Dataset  
Source: UCI Machine Learning Repository

Dataset link:
https://archive.ics.uci.edu/ml/datasets/Breast+Cancer+Wisconsin+(Diagnostic)

The dataset contains diagnostic features such as:

- radius
- texture
- perimeter
- area
- smoothness
- compactness
- concavity
- symmetry

These features are used to train classification models.

## Methodology

The workflow of the project includes:

1. Data loading and preprocessing
2. Exploratory data analysis
3. Feature selection
4. Model training
5. Model evaluation

Machine learning models were implemented using the Python library Scikit-learn.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- tensorflow

## Results

The trained model was evaluated using standard classification metrics such as:

- Accuracy
- Confusion matrix
- Classification report

The experiment demonstrates the potential of machine learning methods for analysing biomedical diagnostic data.
